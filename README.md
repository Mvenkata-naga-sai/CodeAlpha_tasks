🎮 Hangman Game - CodeAlpha Task

📌 Project Description
The Hangman Game is a fun and interactive text-based Python application where the player tries to guess a hidden word one letter at a time. The game randomly selects a word from a small predefined list and challenges the player with a limit of six incorrect guesses.

✨ Features
-  Random word selection from a predefined list  
-  Letter-by-letter guessing system  
-  Input validation for correct entries  
-  Duplicate letter detection  
-  Maximum of 6 wrong attempts  
-  Win and Lose conditions  
-  Simple console-based interaction  

🛠️ Technologies Used
 
 🐍 Python  
 
📚 Concepts Applied
-  Random Module  
-  While Loop  
-  Conditional Statements (if-else)  
-  Strings Handling  
-  Lists
