# 🎮 Hangman Game (CodeAlpha Task)

## 📌 Description
This is a simple text-based Hangman game developed using Python. The player guesses a hidden word one letter at a time.

## 🚀 Features
- Random word selection from predefined list
- Letter-by-letter guessing
- Input validation
- Maximum 6 incorrect attempts
- Win/Lose conditions

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Download the project
2. Open terminal
3. Run:
   ```bash
   python hangman game.py
