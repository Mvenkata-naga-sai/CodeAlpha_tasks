# 🤖 Simple Python Chatbot

A basic rule-based chatbot built using Python `while` loop and conditional statements. It responds to simple user inputs using predefined conditions.

---

## 📌 Features

- Responds to basic greetings  
- Introduces itself  
- Ends conversation with a goodbye command  
- Handles unknown inputs with a default response  

---

## 🧠 How It Works

The chatbot runs in an infinite loop using `while True`.  
It takes user input, converts it to lowercase, and matches it using simple `if-elif` conditions.
