# 📈 Stock Portfolio Tracker

## 📌 Project Description
The Stock Portfolio Tracker is a simple Python application that helps users calculate their total investment in stocks. Users enter stock names and quantities, and the program uses predefined stock prices to compute investment values.

---

## 🎯 Goal
Build a basic stock tracking system that:
- Calculates total investment value  
- Uses manually defined stock prices  
- Displays individual and total investments  

---

## 🚀 Features
- User inputs stock names and quantities  
- Uses a hardcoded dictionary for stock prices  
- Calculates investment for each stock  
- Displays:
  - Stock Name  
  - Quantity  
  - Price  
  - Investment Value  
- Calculates total investment  

---

## 🧠 Concepts Used
- Python Dictionary  
- Input / Output  
- Loops  
- Conditional Statements  
- Basic Arithmetic  

---

## 🛠️ Technologies Used
- Python  

---

## 📊 Example (Individual Stock Calculation)

Stock: RELIANCE  
Quantity: 2  
Price: 2800  

Investment Value = Quantity × Price  
                 = 2 × 2800  
                 = 5600
